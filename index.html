✓ What this course is

The real terminal workflow: git from the command line, branching, PRs via gh, merge-conflict resolution, Actions and CI you author yourself, releases, and branch protection. Everything a builder needs to run a repo the way engineers do.

✗ What it is not

Not the browser-only basics - those live in the non-tech track. No Copilot / AI pair-programming (AI-literacy course, coming). No CodeQL / security scanning (security course, coming). Boundaries kept on purpose.
